Q1 2026 Grant Program

2 de marzo de 2026 Aleksandr Prokudin (169)Announcement (94)FPA (44)Grant (22)166 palabras1 min

The FreeCAD Project Association is now accepting grant applications for the first quarter of 2026. If you are interested in participating, please submit your application by March 10. We will announce the results by March 31.

Please note that we’ve changed the program rules in several ways:

  • Total budget allocation is now 10K EUR per quarter rather than 20K EUR.
  • We now allow only one grant application per developer per quarter.
  • We now prioritize documentation, testing, and improving the release cycle.
  • The technical grant review committee has been dissolved.
  • A “no” vote on a grant now requires a statement from the voter about why they are voting that way, and that feedback will be provided to applicants in an anonymized manner.

The rest of the program rules still apply.

We also still have approx. 20K EUR left in the Ondsel Onwards Fund. If you are interested in working on the Ondsel Lens server or the FreeCAD addon, we encourage you to apply for a grant.

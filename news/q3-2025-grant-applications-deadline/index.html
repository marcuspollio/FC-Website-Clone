The Q3 2025 Grant Applications Deadline Is Near

August 7, 2025 Aleksandr Prokudin (169)Announcement (94)FPA (44)Grant (22)180 words1 min

We will soon wrap up accepting grant applications for the third quarter of 2025. If you are interested in participating, please submit your application by September 1. Any applications submitted after the deadline will be reviewed in the fourth wave in November/December. For general information about the grant program, please refer to this handbook page.

Please note that our budget for the third quarter is $20,000. There may be some competition for the funds. Proposals with a clearly defined scope and realistic price/time estimations have a better chance of being funded.

The program is not restricted to code submissions: you can update the documentation, improve the project’s infrastructure, enhance the parts library, etc.

Code submissions are not restricted to either new features or bugfixes: you can contribute either of those or even both, as is the case with multiple contributors who have already received grants from the FreeCAD Project Association. That said, the developers team is considering a feature freeze around September, so please be aware that active code review of new features will likely resume in 2026.

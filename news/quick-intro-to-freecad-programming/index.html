Quick Intro to FreeCAD Programming

July 25, 2024 Aleksandr Prokudin (169)Event (22)Code (4)73 words1 min

Ajinkya Dahale and Yorik van Havre recently did a live introduction to FreeCAD programming for participants of FOSS Hack 2024, an upcoming hackathon that aims to promote conribution to FOSS software.

The video covers various topics like FreeCAD modules, building the program from source code, a walk through the source code structure, writings tests, etc.

The event is taking place on 27th & 28th July, 2024, offline in India and online worldwide.

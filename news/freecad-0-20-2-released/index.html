FreeCAD 0.20.2 Released

January 4, 2023 Chris Hennes (80)Announcement (94)Release (16)1 min

(Fireworks photo courtesy of Andreas Weith via Wikimedia Commons)

The FreeCAD team is happy to announce the release of FreeCAD 0.20.2!

This is a maintenance release of the FreeCAD 0.20 series. The full list of bugfixes can be found here: Changelog

Installers for MacOS and Windows as well as a Linux AppImage can be found in the assets of the release page at GitHub (at the very last of the page): 0.20.2 Tag

For how to install, see the installation instructions: Installation Instructions

If you use FreeCAD on a Chromebook, please have a look at the known issues sections of the release notes: Known Issues

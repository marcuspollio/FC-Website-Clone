The FreeCAD team is happy to announce the release of FreeCAD 0.20.2!

This is a maintenance release of the FreeCAD 0.20 series.
The full list of bugfixes can be found here:
[Changelog](https://github.com/freecad/freecad/releases/tag/0.20.2#changelog)

Installers for MacOS and Windows as well as a Linux AppImage can be found in the assets of the release page at GitHub (at the very last of the page):
[0.20.2 Tag](https://github.com/freecad/freecad/releases/tag/0.20.2)

For how to install, see the installation instructions:
[Installation Instructions](https://github.com/freecad/freecad/releases/tag/0.20.2#installation_instructions)

If you use FreeCAD on a Chromebook, please have a look at the known issues sections of the release notes:
[Known Issues](https://github.com/freecad/freecad/releases/tag/0.20.2#known_Issues)

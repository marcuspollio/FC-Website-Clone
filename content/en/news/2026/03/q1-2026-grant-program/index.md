The FreeCAD Project Association is now accepting grant applications for the first quarter of 2026. If you are interested in participating, [please submit your application](https://github.com/FreeCAD/FPA-grant-proposals/issues) by March 10. We will announce the results by March 31.

Please note that we've changed the program rules in several ways:

- Total budget allocation is now 10K EUR per quarter rather than 20K EUR.
- We now allow only one grant application per developer per quarter.
- We now prioritize documentation, testing, and improving the release cycle.
- The technical grant review committee has been dissolved.
- A "no" vote on a grant now requires a statement from the voter about why they are voting that way, and that feedback will be provided to applicants in an anonymized manner.

The rest of the [program rules](https://fpa.freecad.org/handbook/process/grant-program-rules) still apply.

We also still have approx. 20K EUR left in the Ondsel Onwards Fund. If you are interested in working on the [Ondsel Lens](https://freecad.github.io/lens-docs/) server or the [FreeCAD addon](https://github.com/FreeCAD/Ondsel-Lens-Addon), we encourage you to apply for a grant.
